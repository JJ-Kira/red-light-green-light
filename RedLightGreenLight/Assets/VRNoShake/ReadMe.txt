Description and purpose:

It uses algorithms to reduce jitter and can customize strength. Appropriate parameters can effectively reduce jitter and provide extremely low delay.

If you are developing a shooting game, you must be troubled by the jitter of the sniper gun when aiming and the weight of the weapon. This script can effectively solve this problem!

If you are developing a cold weapon combat problem, this script can make your weapon not appear to be waving air.

In a word, as long as you are a VR game, you must solve this problem, otherwise the game feel will not be improved.


Usage method:

Hang the UseKalman and UseKalmanRotate scripts on the original data object, and specify the output object Target, which is the effect after output. Note that Target cannot be self owned.

Please check the first demo and try to adjust the MaxV parameter. You can see the anti shake performance and see how the script works.

Please use the second demo, which is based on SteamVR Plugin. You can use different weapons according to the text prompts in the scene, and turn on or off the algorithm to view the use effect.
